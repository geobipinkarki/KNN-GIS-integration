 KNN-GIS-integration
This repository contains Jupyter notebooks developed for my research in geotechnical engineering that demonstrate machine learning applications for predicting soil properties.


1. KNN-GIS Framework for Bearing Capacity Prediction
- **Related publication:** Karki & Manandhar (2026), *Smart Construction and Sustainable Cities*
- **Description:** Implements K-Nearest Neighbors integrated with GIS to predict shallow foundation bearing capacity in complex geological terrain (Pokhara Valley, Nepal)
- **Dataset:** 410 boreholes
- **Key libraries:** pandas, scikit-learn, numpy, matplotlib
- **Highlights:** Achieved R² = 0.997 with optimized k=4 model


 2. Soil-Cement Column Strength Prediction
- **Related publication:** Karki et al. (2023), *Smart Geotechnics for Smart Societies*
- **Description:** Predicts undrained shear strength of soil-cement columns using Multiple Linear Regression, Random Forest, and ANN
- **Dataset:** 92 samples from Bangkok soft clay
- **Key libraries:** scikit-learn, tensorflow/keras, pandas
- **Highlights:** Random Forest achieved R² = 0.99 (train) and 0.94 (test)

- 🛠️ Requirements
- Python 3.8+
- Libraries: pandas, numpy, scikit-learn, matplotlib, geopandas, tensorflow (for ANN)
